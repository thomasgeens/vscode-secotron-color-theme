# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you have security concerns regarding this project, please email thomas@geens.be.

Include as much detail as possible: steps to reproduce, tools and versions used, and any relevant output.
